## üìä The Stats Integration

To ensure the GitHub statistics on the portfolio are always accurate and include my private contributions, I am hosting a custom instance of the `github-readme-stats` API.

* **API Host:** Vercel (Self-hosted fork)
* **Endpoint:** `https://github-readme-stats-dsw3.vercel.app/`
* **Features:** * Bypasses public API rate limits.
    * Includes statistics from private repositories (`count_private=true`).
    * Custom Tokyonight theme configuration.
